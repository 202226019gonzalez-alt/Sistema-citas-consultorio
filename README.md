# SISTEMA DE CITAS MEDICAS
## Descripción
Este proyecto consiste en el desarrollo de un sistema web para la adinistración de citas en un consultorio medico pequeño.  
Su finalidad es optimizar el control de la información relacionada con pacientes, médicos y citas, permitiendo una organización más eficiente.  
El sistemaperite:  
- Registrar pacientes.
- Registrar Médicos.
- Agendar citas.

- **Eliminar**: El sistema nos permite la eliminación de registros.

Para la tabla de Médicos
- **Crear**: El sistema nos permite registrar nuevos pacientes.  
- **Leer**: El sistema nos ermite consultar pacientes registraddos.
- **Eliminar**: El sistema nos permite la eliminación de registros.

Para la tabla de citas

- **Leer**: El sistema nos ermite consultar pacientes registraddos.
